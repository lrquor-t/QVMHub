# 安全漏洞缓解工具

本目录包含针对各类已知安全漏洞的检测和缓解工具。

## 目录结构

```
security/
├── cve-2026-53359/    # Januscape KVM 虚拟化漏洞缓解工具
│   ├── auto-fix.sh    # 自动检查并修复
│   └── README.md      # 详细使用说明
└── README.md          # 本文件
```

## 可用工具

### CVE-2026-53359 (Januscape)

**漏洞描述**: Linux KVM 虚拟化内存管理漏洞，存在长达 16 年，允许虚拟机逃逸到宿主机。

**影响范围**: Linux 内核 2.6.36 - 修补前的所有版本

**快速使用**:
```bash
cd security/cve-2026-53359
sudo ./auto-fix.sh
```

**详细文档**: [cve-2026-53359/README.md](cve-2026-53359/README.md)

## 使用建议

1. **定期检查**: 建议定期运行检查脚本，确保系统安全
2. **测试环境**: 在生产环境部署前，建议先在测试环境验证
3. **备份数据**: 执行任何系统级操作前，请确保重要数据已备份
4. **阅读文档**: 使用工具前请仔细阅读对应的 README.md 文档

## 注意事项

⚠️ **重要提醒**:
- 所有脚本需要 root 权限运行
- 某些操作可能需要重启系统或重新加载模块
- 部分缓解措施可能影响现有功能，请仔细阅读文档
- 缓解工具只是临时措施，永久解决方案是升级到已修补版本
